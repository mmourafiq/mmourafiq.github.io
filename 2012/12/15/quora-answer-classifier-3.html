Quora Answer Classifier (Part 3, linear and non linear classifiers).

Designing an algorithm to develop classifier that is able to tell good answers from bad answers, as well as humans can.

We saw in previous posts how we could use decision trees and K-nearest neighbors to classify answers.

In this post, the last of this series of posts, we will be using two classifiers; linear and non linear classifiers.

Grab the code of the model item from part 1. Since we are going to use it for this two classifiers.

Linear Classifiers

It is one of the simplest classifiers to construct. It works by finding the average of all the data in each class and constructing a point that represents the center of the class. It can then classify new points by determining to which center point they are closest.

First we need a function that calculates the average of items. Then we need a way to calculate the closeness of new item. While we can use Euclidean distance, which will work for this classifier, we can take a different approach using vectors and dot-products. Also we will be using the scaling method, since all the variables arenâ€™t really comparable.

 1 def linear_mean(data, t_value, f_value, mask=None):
 2     #two categories
 3     t_coords = []
 4     f_coords = []
 5     for item in data:
 6         if item.value == t_value:
 7             if mask is None:
 8                 t_coords.append(item.scaled_coords)
 9             else:
10                 t_coords.append([item.scaled_coords[i] for i in xrange(len(item.scaled_coords)) if i in mask])
11         elif item.value == f_value:
12             if mask is None:
13                 f_coords.append(item.scaled_coords)
14             else:
15                 f_coords.append([item.scaled_coords[i] for i in xrange(len(item.scaled_coords)) if i in mask])
16     t_mean = numpy.average(numpy.array(t_coords), 0)
17     f_mean = numpy.average(numpy.array(f_coords), 0)
18     return t_mean, f_mean
19 
20 def dot_product_classification(item, means, t_value, f_value, mask=None):
21     t_mean, f_mean = means
22     if mask is None:
23         item_coords = numpy.array(item.scaled_coords)
24     else:
25         item_coords = numpy.array([item.scaled_coords[i] for i in xrange(len(item.scaled_coords)) if i in mask])
26     y = (numpy.dot(f_mean, f_mean) - numpy.dot(t_mean, t_mean)) / 2
27     x = (numpy.dot(item_coords, t_mean) - numpy.dot(item_coords, f_mean)) + y
28     return t_value if x > 0 else f_value

Non Linear Classifiers

We will be using a very basic non linear classifier. also we will be using the radial-basis function as a kernel trick for our classifier. which replaces the dot-product function, returning what the dot-product would have been if the data had first been transformed to a higher dimensional space using some mapping function.

 1 def radial_basis(item1, item2, gamma=10):
 2     diff_item = numpy.array(item1.coords) - numpy.array(item2.coords)
 3     dist_item = numpy.sqrt(numpy.sum(diff_item))
 4     return math.e ** (-gamma * dist_item)
 5 
 6 def non_linear_classification(item, data, t_value, f_value, offset, gamma=10):
 7     t_sum = 0
 8     t_count = 0
 9     f_sum = 0
10     f_count = 0
11     for item2 in data:
12         if item2.value == t_value:
13             t_sum += radial_basis(item2, item, gamma)
14             t_count += 1
15         elif item2.value == f_value:
16             f_sum += radial_basis(item2, item, gamma)
17             f_count += 1
18     res = (float(t_sum) / t_count - float(f_sum) / f_count) + offset
19     return t_value if res > 0 else f_value
20 
21 def calculate_offset(data, t_value, f_value, gamma=10):
22     t_data = []
23     f_data = []
24     for item in data:
25         if item.value == t_value:
26             t_data.append(item)
27         elif item.value == f_value:
28             f_data.append(item)
29     t_sum = sum(sum([radial_basis(item1, item2, gamma) for item1 in t_data]) for item2 in t_data)
30     f_sum = sum(sum([radial_basis(item1, item2, gamma) for item1 in f_data]) for item2 in f_data)
31     return (float(f_sum) / (len(f_data) ** 2)) - (float(t_sum) / (len(t_data) ** 2))
